Executive Summary
一句话结论：图片不只是「被理解」，而是「撬动价值链的输入模态」。灵光应以感知变现为短期引擎、金融信息翻译为垂直锚点、独立Agent产品为中长期方向，三线并进。

研究全景

本报告调研了国外5大（ChatGPT、Claude、Gemini、Perplexity、Grok）和国内10大（豆包、Kimi、通义千问、文心一言、智谱清言、讯飞星火、腾讯元宝、DeepSeek、MiniMax、百川）AGI应用在「上传图片」功能上的能力全景，覆盖感知提取、分析推理、视觉搜索、知识增强、决策辅助、工作流自动化、Agent行动、实时交互八大能力域。

三大核心洞察

1
双轨路线：感知变现 + 行动探索
图片输入价值重心正从「看懂」迁移到「触发行动」（Claude Computer Use 6个月5倍增长、豆包手机秒光），但行动层全球无商业化验证，感知层已有盈利模型。灵光应「感知变现养团队 + 行动探索布局未来」，资源配比70:30。
2
垂直锚点：拍照看懂金融
定位「金融信息翻译器」而非「金融顾问」——帮用户看懂保险条款、拆解理财产品参数、对比收益说明书，只做信息整理+对比展示，不输出推荐/建议。首个标杆场景：理财产品对比。
3
独立Agent：灵光自主发展
灵光作为独立Agent产品自主发展，以图片输入为核心交互入口，蚂蚁系能力以API形式接入而非依赖BU嵌入，绕开组织协调瓶颈。

研究背景与方法

研究背景

2025-2026年，多模态AI进入爆发期。全球主流AGI应用纷纷将图片输入作为核心交互方式，图片不再只是「被看懂的对象」，而是「撬动价值链的输入模态」——从OCR识别到Agent自动执行，图片输入正在重塑用户与AI的交互方式。

灵光App作为蚂蚁集团旗下AI应用（周活295万，国内原生AI App Top 10），正处于多模态能力规划的关键窗口期。本报告旨在回答：图片作为输入模态，能撬动哪些价值链？灵光的差异化切入点在哪？

研究框架

主框架：JTBD（Jobs To Be Done）— 从用户任务视角构建分析骨架
增强框架：竞争定位图（可视化能力矩阵）+ 蓝海战略画布（差异化空间）+ 三层面增长（H1/H2/H3分层）
可选框架：PESTEL（政策与技术趋势影响）

能力域定义

本报告将图片输入触发的能力分为八大域，从基础感知到高阶行动，形成完整的价值阶梯：

层级能力域典型场景价值定位
L1 基础层感知提取OCR、物体识别、场景理解数据入口
L2 分析层分析推理图表解读、医疗影像、多图对比专业增值
L3 检索层视觉搜索以图搜图、同款识别连接信息
L4 知识层知识增强拍照问答、实物科普、题目解析教育赋能
L5 决策层决策辅助商品比价、健康评估、穿搭建议辅助判断
L6 工作流层工作流自动化截图→代码/PPT/文档效率倍增
L7 行动层Agent行动UI操控、跨应用执行替代操作
L8 实时层实时交互摄像头流式分析、AR叠加即时感知

核心洞察与战略建议

三大核心洞察（Tier 1, 18-20分）

P0 核心洞察

洞察1：双轨路线 — 感知变现 + 行动探索

现象：Agent/工作流自动化是增速最快的图片能力域（Claude 5倍增长、豆包手机秒光）

含义：行业价值重心从「理解图片」向「图片触发行动」迁移

但是：行动层商业化路径全球未验证，感知层已有成熟变现模型（扫描全能王/Mathpix已盈利）

行动：灵光应「感知变现（OCR/文档/图表）养活团队 + 行动探索（Agent/工作流）布局未来」，资源配比70:30

红队修正：原建议「优先投资行动」过于激进。怀疑派投资人指出「增速最快≠商业化最快」，修正为双轨制

P0 核心洞察

洞察2：垂直锚点 — 拍照看懂金融

现象：垂直深耕策略已被验证（百川医疗HealthBench全球第一，讯飞教育），通用追赶不如垂直突破

定位：「金融信息翻译器」而非「金融顾问」— 帮用户看懂保险条款、拆解理财产品参数、对比收益说明书

红线：产品绝不输出「推荐/建议」，只做信息整理+对比展示。监管者指出「A比B好」即构成投资建议，需持牌

首个场景：理财产品对比 — 用户截图/拍照理财产品页面→AI抽取收益率、风险等级、期限、起购金额→结构化并列展示

准确率警示：模型在金融文档上准确率约92-95%，剩余5-8%在金融场景是灾难级。初期必须「AI+人工确认」混合模式

P0 核心洞察

洞察3：独立Agent产品路线

现象：纯视觉GUI Agent正引发「OS入口之争」，豆包手机被微信/美团/阿里集体封杀说明威胁等级极高

战略：灵光作为独立Agent产品自主发展，不依赖支付宝BU授权。以图片输入为核心交互入口

路径：蚂蚁系能力以API形式接入而非依赖BU嵌入，绕开组织协调瓶颈

天花板：独立路线受限于灵光自身DAU（295万周活），需Agent体验本身驱动增长

四大战略建议（SMART格式）

B1

295万周活的用户画像和需求差异未做分层分析。一二线用户vs下沉用户的图片需求差异可能很大。
